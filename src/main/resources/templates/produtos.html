CRUD de Produtos em Spring Boot

Todas as rotas do CRUD

  • Método GET em /produtos/todos, para trazer todos os produtos: /produtos/todos
  • Método GET em /produtos/produto?id=numeroId, para trazer um produto específico: /produtos/produto?id=numeroId

    • É necessário passar um número no lugar do 'numeroId'

  • Método DELETE em /produtos/excluir, para remover um produto específico: /produtos/excluir

    • É necessário passar um JSON como o formato:

    Ex: { "id" : 1 }

  • Método POST em /produtos/adicionar, para adicionar um produto: /produtos/adicionar

    • É necessário passar um JSON como o formato: (descrição é opcional)

    Ex: { "nome": "Banana", "descricao": "Fruta", "preco": 11.59 }

  • Método PUT em /produtos/atualizar, para atualizar produto específico: /produtos/atualizar

    • É necessário passar um JSON com o formato: (descrição é opcional)

    Ex: { "id": 2, "nome": "Amora", "descricao": "Fruta", "preco": 11.59 }
